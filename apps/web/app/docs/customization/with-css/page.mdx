# With CSS

A guide to customizing the styles of the menu with CSS.

## Using Default Styling

This library comes with a default stylesheet which you can import using:

```tsx:layout.tsx
import 'kmenu/cmdk.css'
export default () => /* ... */
```

### Customizing Root Variables

After you import the stylesheet, you must add in some root variables. You can use the default variables by importing:

```tsx:layout.tsx
import 'kmenu/vars.css'
import 'kmenu/cmdk.css'
export default () => /* ... */
```

You can customize nearly all parts of this menu with [CSS root variables](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties) without having to touch things like positioning and alignment. Here are the default root variables:

  --breadcrumb-margin: 5px 0px 0px 5px;
  --breadcrumb-padding: 5px 10px;
  --breadcrumb-border: 1px solid #e2e2e2;

  --command-font-size: 14px;
  --command-icon-size: 16px;

  --command-inactive-color: hsla(0, 0%, 51%, 1);

  --command-margin: 0px 0px 0px 14px;

  --command-heading-font-size: 12px;
  --command-heading-color: #666666;
  --command-heading-margin: 5px 0 10px 10px;

  --shortcut-inactive-color: hsla(0, 0%, 51%, 1);
  --shortcut-active-color: hsla(0, 0%, 20%, 1);

## Style it Yourself

You can copy the style file(s) from [here](https://github.com/haaarshsingh/kmenu/blob/master/apps/examples/styles/empty.css), and copy paste it and import into the your project to customize the styling. Alternatively, I'll leave a blank file here if you want to customize it from scratch:

```css:cmdk.css

}

.dialog {
}

.crumbs {

}

.command_wrapper {
}

.command {
}

.shortcuts > kbd {
}

.shortcuts {
}

.info_wrapper {
}

.info_wrapper > svg {
}

.category_header {
}

.selected {
}

.command_text {
}

.scroll_ref {
}

  .dialog {
  }
}
```
