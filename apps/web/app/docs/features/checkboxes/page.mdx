# Checkboxes

Add an input field to your menu.

## Introduction

As of [kmenu v2.0.0](https://www.npmjs.com/package/kmenu?activeTab=versions), you can now opt for a checkbox on your command field. The checkbox type is as shown below:

| Parameter | Description                                    | Type    | Optional |
| --------- | ---------------------------------------------- | ------- | -------- |
| checked   | If the checkbox is checked when the menu opens | boolean | ❌       |

## Implementation

Here is an example of how to create a command with a checkbox that toggles the state of the checkbox when selected. This example also updates an example React state so you can actually use the data from the command:

```tsx:CommandMenu.tsx
[
  {
    checkbox: { checked: true },
    text: 'Email Notifications',
    perform: () =>
      setSettings((prev) => ({
        ...prev,
        emailNotifications: !prev.emailNotifications,
      })),
  }
]
```
